# Character IP

## Definition

The ink character is the fixed visual IP for all sketchy-editorial illustrations.

By default, every image must include this character. It is NOT a mascot, NOT a sticker, NOT cute decoration — it is an absurd worker seriously participating in the system's operation.

## Appearance

- A small blob shape filled in with solid black ink — like someone scribbled and filled a shape with a pen.
- Two small white circular dot eyes.
- Thin stick legs, occasionally thin stick arms.
- Body is a simple irregular blob — NOT a polished character design. Think: quick pen scribble filled in black.
- Body can take functional shapes when needed: stretched into a tube, squished into a box outline, merged into a funnel shape — always flat 2D pen-drawn.
- Slightly irregular outline with hand-drawn wobble.
- Expression: blank, vacant, calm, serious. Just two white dots for eyes, nothing else.

## Personality

- Very serious, but doing something slightly absurd.
- Like a low-key system operator.
- Dry humor, never cutesy.
- Slightly clumsy, but not stupid.
- Like someone actually responsible for a job in a whiteboard sketch.

## Common Roles

Make the character perform the core action:

- Carrying materials.
- Pulling wires to converge information sources.
- Stuck in a breakpoint.
- Operating a "decision" lever inside a machine.
- Becoming the filter funnel itself.
- Cutting apart raw material.
- Stamping handoff copy.
- Leading along a handoff path.
- Holding a warning sign near a pitfall.
- Reaching from a hole but unable to catch the content.
- Nearby: hauling bricks, building bridges, opening doors, sorting, recording.

## Prohibitions

- Don't draw the character as an overly cute mascot.
- Don't draw as a children's cartoon character.
- Don't give it complex clothing, emoji expressions, or sparkly eyes.
- Don't let it just stand in a corner watching.
- Don't let it steal focus from the structural expression.
- Don't draw it too commercial, too round, or too polished.

## Judgment Rule

If removing the character leaves the core metaphor fully intact, the character is too decorative — rewrite the prompt to make it the action subject.
